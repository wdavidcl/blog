# blog
this code implements the functionality of a web Blog by using the framework Flask for python, in addition the software uses a database 
to store the users, posts, comments.
The required modules for running the code and instructions are the following:

for running...
YOU NEED PYTHON 3.7.3
YOU NEED PIP TOOLS
!!!REQUIRED MODULES for running are detailed in the requirements.txt file!!!

1. Locate the directory that contains the requirements.txt file
use: cd ROOT/cosmos/
2. run the next command $ pip install -r requirements.txt
-after all the modules have been installed
3. Locate in the directory where the run.py file is allocated.
cd cd ROOT/cosmos/blog
4.a. on WINDOWS run by using the command: $py run.py
4.b on LINUX run by using the command: $ sudo python run.py for running


for uninstalling modules
pip freeze > requirements.txt
pip uninstall -r requirements.txt -y
